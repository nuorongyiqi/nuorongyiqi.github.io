发表于 2019-07-15

命令结构：
命令 【-选项】 【参数】

  • 在centos 7中 ， 右击打开终端

代码 含义
mkdir 创建目录 （ mkdir DIRNAME）
ls 显示目录中的文件
touch 创建文件
rm 删除文件（不能删目录）
cd 切换目录
rm -f 强制删除
em -r 删除目录
cd 切换目录
yum list 显示软件列表
yum install APPNAME 安装指定软件
yum search APPNAME 搜索相关软件
yum check-update 检查是否需要更新
yum update APPNAME 更新指定软件
yum update 更新所有软件
yum remove APPNAME 卸载指定软件
